## How to Add More Posts

### Method 1: Direct LinkedIn Access
1. Visit Dharmesh Shah's LinkedIn profile: https://www.linkedin.com/in/dharmesh
2. Find the post you want to save
3. Copy the full text
4. Create a new markdown file in this folder
5. Use the template below

### Method 2: Using Browser Tools
1. Open the LinkedIn post in your browser
2. Use "Save Page As" to save as HTML
3. Or copy text and create markdown file

### Method 3: Screenshot
1. Take a screenshot of the post
2. Save as image file
3. Reference in markdown file

## File Naming Convention

- `post-[number]-[topic].md` for markdown files
- `post-[number]-[topic].html` for HTML saves
- `post-[number]-[topic].png` for screenshots

Example: `post-04-idea-gluttony.md`
